B: Objects Tab Active

2×2 grid of draggable object types at top, list of placed objects below with dimensions. Drag ghost shows on canvas during drag.

Interaction Flow

  • Drag from catalog → ghost preview follows cursor → drop on canvas with snap assist
  • Click object on canvas → highlights with blue border → sidebar auto-switches to Configure tab
  • Click object in sidebar list → same as clicking on canvas (selects + switches tab)
  • Click Objects tab manually → back to catalog view, deselects current object
