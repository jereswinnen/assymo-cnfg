Resize Handle Style

How should the draggable edges look when a building is selected in the 2D schematic?

A. Edge highlight

Edges highlight when hovered, cursor changes to resize arrow. Clean, minimal — no visible handles until you hover.

B. Midpoint grab handles

Small circles at the midpoint of each edge, always visible when selected. Clear affordance that edges are draggable.

C. Thick selection border + corner dots

Thicker border on selected building with corner markers. Entire edge is the drag target. Familiar from design tools like Figma.
