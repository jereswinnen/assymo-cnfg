Which views should support drag-to-resize?

Your app has a 2D schematic view and a 3D view. Where should edge-dragging work?

A

2D Schematic only

Drag edges in the top-down schematic view. Simpler to implement — edges map directly to screen coordinates. Consistent with existing drag-to-move which is also 2D only.

B

3D View only

Drag edges directly on 3D buildings. More immersive but harder — requires raycasting to detect edge proximity and project mouse movement onto the correct axis.

C

Both views

Drag-to-resize works in both 2D and 3D. Most flexible, but more work. Could start with 2D and add 3D later.
